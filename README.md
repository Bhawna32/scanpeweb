# Scanpe - Biometric Palm Payment System

A modern, minimalist website for Scanpe, a biometric palm payment system. Built with React, TypeScript, and Tailwind CSS, featuring a high-tech design inspired by Keyo's aesthetic.

## Features

- **Full-width hero section** with "Pay with Your Palm" headline
- **Minimalist navigation** with smooth scroll and anchor links
- **Dotted grid background** pattern for visual appeal
- **Bold typography** using Inter and Poppins fonts
- **Subtle animations** including fade-in effects and hover states
- **Responsive design** that works on all devices
- **Modern UI components** with gradient accents and clean layouts

## Design Elements

- Light dotted grid pattern background
- Dark text on light background with subtle gradients
- Bright accent colors (neon-blue, purple) for CTAs and hover states
- Large, bold sans-serif typography
- Smooth transitions and hover animations
- Clean, minimalist layout

## Getting Started

1. Install dependencies:
   ```bash
   npm install
   ```

2. Start the development server:
   ```bash
   npm start
   ```

3. Open [http://localhost:3000](http://localhost:3000) to view it in the browser.

## Built With

- React 18
- TypeScript
- Tailwind CSS
- Modern CSS animations and transitions

## Sections

- **Hero**: Full-width section with main headline and CTAs
- **How It Works**: Three-step process explanation
- **Features**: Six key features with hover animations
- **Contact**: Contact form and company information
- **Footer**: Links and company details

## Customization

The design uses Tailwind CSS with custom colors and animations defined in `tailwind.config.js`. You can easily customize:

- Colors in the theme configuration
- Typography settings
- Animation timings and effects
- Component styles in `src/index.css`
